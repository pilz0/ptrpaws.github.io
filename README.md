# Jekyll Blog
---

Simple and minimal Jekyll blog. 
Forked from [Asko](https://github.com/manuelmazzuola/asko).

Now deployable on Github Pages, with:

### Features

- Responsive Design
- Dark/Ligh theme
- Inline CSS
- Anchor headings
- Tags & Tag pages
- 404 page
- Robots.txt
- Atom & Json feeds
- Sass
- Ultra fast (Only ~5kb of CSS)

Feel free to fork!
